# Reading Rainbow Bookstore Database

This project is a SQL Server relational database designed for a bookstore system. It includes tables for branches, books, publishers, authors, inventory, sales, and employees.

## Project Overview

The goal of this project was to design and implement a relational database that can store bookstore data, manage inventory, track sales, and organize information about books, authors, publishers, and branch locations.

## Technologies Used

- SQL Server
- T-SQL
- Relational Database Design
